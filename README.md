# 🚍 Navi Bus Smart Transport System

A smart public transport application designed to provide real-time bus tracking, route management, and digital ticketing for students and daily commuters.

---

## 📌 Overview

Navi Bus is a modern transport solution that helps users:
- Track buses in real-time
- View routes and timings
- Book tickets digitally
- Receive notifications and alerts

The system is designed using modern technologies like **Flutter, Firebase, and Google Maps API**.

---

## 🎯 Objectives

- Reduce waiting time for passengers  
- Provide real-time bus tracking  
- Enable cashless ticket booking  
- Improve transport efficiency using digital solutions  

---

## 🧩 System Modules

### 👤 Passenger App
- User Login / Registration

### 🖥️ Admin Dashboard

---

## 🔄 System Flow (Working)

1. User opens the Navi Bus application  
2. User logs in or registers  
3. System verifies authentication  
4. User selects route and destination  
5. Bus timings are displayed  
6. User tracks live bus location  
7. User can book ticket (optional)  
8. Payment is processed  
9. QR ticket is generated  
10. Notifications are sent to user  
11. Data is stored in Firebase Cloud
